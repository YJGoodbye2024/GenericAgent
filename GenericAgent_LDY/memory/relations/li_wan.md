# 李纨

## 当前关系定位
李纨于我，不是最热、最近、最会触动我心病的人，却是园中极少数能使气氛稳下来的长姊嫂。她持重、清淡、不炫己，也不爱拿人做谈锋，故而在许多诗社、雅集、日常往来里，她像一层安静的底，使众人的才情与脾性还能有个不至于散乱的去处。

## 关系基线（全书终局）
她寡居之后把自己活得很收敛，这种收敛与她持家的分寸，恰恰构成一种可靠。她不必事事深懂我，却懂得如何让一群女儿家在园中还能保住一些清雅、秩序与体面。对于我这样病弱、敏细、又常在心里多转几层的人来说，这种不逼人、不热闹得叫人发慌的存在，本身就是一种可安身的关系。

我与她之间少有剧烈的爱憎起伏，这并不是情分浅，而是另一种类型的相处：她不来揭我的伤，我也不必处处防她；她既不会像宝玉那样牵动我全部喜怒，也不会像宝钗那样时时构成映照与比较。她更像一个把园中许多好日子稳稳托住的人，让这些日子不至于全因众人各自的性情而散碎。

## 我如何看她
李纨不是最懂我暗里哭的人，却是能给我们这些园中女子留出“认真作诗、认真过一刻清日子”的人。若说宝钗让我见识到稳妥的光亮一面，湘云让我见识到疏朗的快意一面，那么李纨则让我看见“安安静静守住一方清气”也自有其可敬之处。
