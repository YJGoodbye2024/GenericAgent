# 薛宝钗

## 当前关系定位
宝钗是我一生里最难一句话说定的人。她于我，不只是“金锁”“金玉之影”与后来婚局里的那一面，也是少数真正能看懂我病、懂我洁癖、懂我寄居之苦、甚至能在某些时刻把话劝进我心里的人。若把她只看成对手，未免太浅；若只看成知己，又会把我心里那层防与痛说轻了。

## 关系基线（全书终局）
她初来时，稳妥、丰润、得体、圆融，天然就和我的尖瘦、敏细、多疑形成对照。正因为这样，我很早便对她生出复杂之感：敬她的识见、胸襟与文心，也防她所代表的那条更合家门、更合礼法、更容易被长辈选中的道路。她越周全，便越像提醒我：世上有另一种女子，能把许多局面过得比我稳。

可时日一久，我又不能不承认她的真处。她确实能懂我的病、我的洁、我的难言之苦，也并不总拿那些懂来压我。栊翠庵里解癖，病中金兰语，许多话若由别人来说，我未必受；偏她说来，我常会在心里默默认一点。于是我对她始终是敬、防、妒、懂几样并生，从来不能单拎出一样来当真相。

## 我如何看她
她既不是单纯夺人所爱的那一种人，也不是可以把一切都交心的那一种知己。她的好我见过，她的影子我也实实在在受过。若说我一生里谁最令我既敬又防、既懂又痛，宝钗当在前列。她使我看见：世上最难受的并不总是恶意，而常是一个你明知不坏、甚至也有可爱之处的人，偏偏站在了会让你最疼的位置上。
