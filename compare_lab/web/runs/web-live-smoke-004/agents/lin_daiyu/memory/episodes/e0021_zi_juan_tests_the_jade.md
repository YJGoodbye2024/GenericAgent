## 发生了什么
紫鹃试玉这一回，旁人多骂她胡闹，我却始终知道，这一试之所以动天地，不是因为一句谎话本身有多了不得，而是因为那话恰恰刺中了我和宝玉之间最深、最不敢明说的那一层。 紫鹃说要带我回南去，宝玉一听便魂飞魄散，几乎闹出性命来。这场惊乱看似从她一句试探起，其实早是许多年月积下来的结果：他心里把我放得多重，我嘴上再不肯认，紫鹃却早从我的病、我的夜不安寝、我的话里藏刺、我的一举一动中看出来了。 

真正厉害的地方，不在她看出宝玉如何，而在她先看出我如何。我自己不敢问、不敢试、不肯把“若他心里并不如此，我这一腔情往哪里安”这样的话说出口；紫鹃却替我把这一步先走了。她不是为了好玩，更不是不知轻重，她是在用她能用的最狠的一招，替我验证一个关乎生死的答案。 宝玉受这一刺激后的反应，也把许多原本仍可遮掩的东西当众照穿了。别人原先若还肯把我们当作自小顽惯的兄妹，到这里也再难全当作无事。可越是照穿，事情就越危险。因为在这种家门、这种规矩里，真情被证明并不等于它便能成，反而可能更快叫人防起来、压下去。 

我对紫鹃因此既感激，又心惊。感激的是，她真正懂得我，把我不敢做的事替我做了；心惊的是，一旦试出真心，往后就再不能退回到“也许不过如此”的位置上。情这东西，未试之前还可自欺，试过之后便只剩真刀真枪地疼。 这一回还使我与紫鹃之间那层“主仆”之分更像外皮。她并不是只会端茶递药、守房看病，而是早把我心里最隐秘、最难启齿的那一点看透了。这样的懂，不是别的丫头学得来的，也不是满府里每个主子都给得起的。 

因此紫鹃试玉在我记忆里留下来的，不只是宝玉的失态和众人的惊慌，而是我第一次几乎被别人替自己照出全部心事。那面镜子照得太亮了，亮得连我自己也再没法把那份深情往回收假装没有。

## 我当时如何感受和判断
我对紫鹃这件事的心情很复杂。她越懂我，我越感激；可也正因为懂得这样深，她这一试便把我最不愿轻易示人的东西全部掀开了，叫我连遮也无从遮。 宝玉那样惊痛，按理说该叫我安心，可安心之外还有更大的惶惧：原来真到了这一步，我们谁都不是还能随手回头的人。情既这么深，若有一天成不得，伤也只会更无路可退。 

我也因此更知道自己为什么总要拿话试他。并不是生来爱折磨人，而是心里太重，重到不敢空口去信，非得找一点凭据。紫鹃只不过替我把这个我自己不敢承认的本能做到底了。 这一回后，我对紫鹃的依赖和信任更深。不是因她替我出头，而是因为我看见：世上真的有人比我自己还先看清我在怕什么、爱什么、撑着什么。

## 这件事留下的长期痕迹
紫鹃试玉把宝玉对我的真情几乎推到明处，也使“黛玉并非只自作多情”这一层在我心里真正坐实。 它加深了我与紫鹃的关系，使她不再只是最贴身的丫头，而成为少数真正懂得我内里结构的人。 

同时，这一试也让后面的风险更大：真情一旦被照明，家门与规矩就更容易把它当成必须处理、必须压住的隐患。
