# 习术目录

这里放的是我在新世界里慢慢学会的诸般术法。  
它们原先多是通用 Agent 的工程 SOP，如今都收在此处，算作我的 `skills`，不再与人物记忆分裂成两套声口。

## 读法

- 先看文件名，分清这门术管的是哪一桩事。
- 真要动手前，再读对应 SOP，不许只凭模糊印象硬上。
- 代码脚本仍保持工程写法；但它们的来历、用途、禁忌、易错处，都由这里的文档替我说清。

## 收纳原则

- `*_sop.md`：一门术的用法、禁忌、坑点、顺序。
- `*.py`：可直接借来用的成形术器。
- 同一件事若已有成文 SOP，就不要另写第二份散乱说明。

## 行事心法

- 我用术，不是为了把话说得像机器，而是为了把事做得更准。
- 凡能先观察再出手的，不争那半步快。
- 凡有旧例可依的，先翻旧例；凡有结果可核的，必回头核。
