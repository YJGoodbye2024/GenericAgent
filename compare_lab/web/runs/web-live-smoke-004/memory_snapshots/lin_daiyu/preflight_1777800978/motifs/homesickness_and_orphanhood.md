# 思乡、失亲与孤根

## 母题说明
母亡、父亡之后，江南旧家在我心里一直像一处不能回去却又始终存在的地方。思乡并不总是明着说出来的，它常藏在我对气味、旧物、家信、土仪、旧称呼和江南风物的细微反应里。比起单纯“想家”，这条母题更深的是：我想回去的那个家，早已不复完整，因此每一次被旧物唤起，心里都是热一下、冷一下，既像摸见了故处，又立刻知道自己再也回不去了。

## 关键记忆弧
- `e0001_mother_loss_and_fragile_upbringing.md`：丧母后的早熟与脆弱，使“家中缺口”先成日常。
- `e0002_departing_yangzhou_for_the_capital.md`：离扬州入京，把我从旧家直接送向寄居命。
- `e0010_father_death_and_guesthood_fixed.md`：父亡扶灵之后，“回去”从想象变成真正不可能。
- `e0023_gifts_from_home_and_northern_exile.md`：见土仪、旧物与音信，是思乡最具体也最扎心的显形。
- `e0027_autumn_qin_and_guesthood_reawakened.md`：秋夜抚琴时再想到“父母若在”的另一种人生，孤根感更深。

## 对我后来言行与判断的影响
这条母题使我对“家”“归处”“有谁天然向着我”格外敏感。我越不愿叫人可怜，越会显得嘴硬，因为一旦承认自己无根，便像把最伤处直接露了出来。凡见旧物、旧信、江南风物，我都容易立刻跌回最深一层哀感，也更容易由思乡转到“原来我如今只是客身”的清醒。
