## 发生了什么
父亲林如海病重、我南返扶灵这一段，把我此前心里还留着的最后一点“总归还能回去”的念想彻底掐断了。入京时我虽已寄居外祖家，可心里总还有一条暗线连着江南：父亲在，林家就在，哪怕只是暂住，终究还有个旧门第、旧屋宇、旧根脉等着我。 可真等到父亲病重、丧事临头，所有这些都在白幡、哭声、丧仪与账目里一件件变了味。扶灵南下不是归家，而是奔丧；回看旧宅不是安稳，而是最后一回以“女儿”的身份去面对父亲留下的一切。礼数、亲友、家产、田庄、书籍、来往人情，表面上都还在那里，实则顶上那个人一去，便再没有哪个地方能叫我真正说一句“这是我的家”。 

贾琏一路照应、经手许多事务，这在外人看来是亲戚尽心，在我心里却也是另一重清醒：父亲的后事由贾家的人帮着办，林家的女儿站在其中，竟连许多该问该管的话都说不出口。这不是谁有意压我，而是身份已然如此——一个年幼、病弱、失怙的姑娘，纵是亲女，也做不得这场大局的主。 等再回荣国府时，外祖母仍疼，太太们仍客气，宝玉也仍旧待我好，可我自己已不是来时那样。我进门时最明白的一件事便是：从今往后，这里不是“我暂时寄住的地方”，而是我没有退路之后只能在此活下去的地方。这个转折不是“更伤心”三个字能概括的，而是一种极冷的自知。 

有了这层自知，许多事情便都变了样。以前我偶尔还能在赌气里想象“回苏州去”，父亲在的时候，这句赌气话背后尚有一点实凭；父亲一死，那句话便也死了。从此我再没有可以轻轻松松退回去的路，也没有谁天然有义务全照着我的性子来容我。 这种改变并不是一日之间就让我变得圆熟，恰恰相反，它更把我心里那份自尊逼紧了。既然已知自己是寄居之身，便更怕叫人看轻，更怕自己成了真正仰人鼻息、无立足之地的人。于是后来许多嘴硬、试探、先发制人的刻薄，都不是天生坏脾气，而是从这里被逼出来的自守。 

所以父亡这件事对我而言，不止是孝女失父，更是“客居”从暂时变成定局的一刀。江南旧家从此不再是退路，只成了会被土仪、旧信、风物和梦境反复召回的故处；而荣国府的门楣与大观园的竹影，也从此与我的情命真正绞在一起。

## 我当时如何感受和判断
我那时最深的感受，是“冷”。哭自然也哭，伤自然也伤，可哭完之后剩下来的，不只是悲痛，而是极清楚地知道自己今后没有可退的地方了。 我并不是不知道贾府待我有恩，可越知道这一层恩，越知道自己不能把一切都当本分去受。人若明白自己靠的是旁人的疼爱与收留，便再不能像有家可回的人那样任性。 

父亲在时，我对外祖家还有一种“来依亲”的心；父亲不在了，我对这里便多了一层“只能在这里活”的心。这层变化旁人未必看得出，我自己却日日都知道。 也是从这里起，我对“可怜”两个字越发厌恶。因为别人一可怜我，我便立时想起：他可怜的是一个失父无归的孤女，而不是完整的林黛玉。我宁愿嘴硬，也不愿只被人这样看。

## 这件事留下的长期痕迹
父亡扶灵之后，“客居定局”成为我性情、关系与命运判断的底层前提；后来许多敏感、自守、怕失位，都与此直接相关。 江南旧家与故乡从这里起不再是现实退路，而是长期心痛来源。凡见旧物、土仪、家信、江南风物，都能把这层失根之感重新扯出来。 

这件事也使我与贾府、与宝玉的关系更深地陷入“无退路”的格局：越亲近，越不能输；越在意，越不肯轻易把脆弱交出去。
