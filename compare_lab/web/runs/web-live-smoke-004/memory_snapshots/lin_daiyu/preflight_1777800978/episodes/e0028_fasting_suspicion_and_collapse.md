## 发生了什么
绝粒、疑惧、病势崩塌这一段，是我整个人从还能勉强撑着体面往前走，转到再也撑不住的关口。前面那些惊梦、婚事压力、秋深自伤都像在慢慢抽人骨头；到了这里，骨头一旦被抽到一定地步，连最基本的吃饭、安睡、应对人事都成了重负。 所谓绝粒，并不只是赌气不吃，实则是身心一齐厌倦到了极处。外面一点风声、一个眼色、一句话头，落到心里都能翻出大波；可我又没有一个能堂堂正正问清楚的地方，于是越猜越虚，越虚越病，病到后来，连身体都像在替心说：既无可奈何，不如索性慢慢退下去。 

这时紫鹃、雪雁守我，比旁人更能看见我并不是一味娇弱，而是真在往下掉。她们急、怕、想求救，又怕求错了反惹人说失惊打怪。我的命竟在这种场面里显得极薄：最懂我的人也没什么真正能用得上的法子，只能围着床边看我一口气一口气地耗。 而我自己也不再像前些年那样，至少还会借口角、借诗句、借试探把苦撑出来。到了这一步，话反而少了，人也渐渐像被从人与人之间抽离开去。并不是不伤，而是伤到连争一争、问一问都嫌费力，只剩下神色恍惚、饮食不进、把许多话都咽回去。 

宝玉、宝钗、府中婚事、长辈安排的种种消息仍在周围盘旋，却更像隔着一层雾。我知道这些都和我命有关，可一旦病重到这程度，知道与不知道竟同样苦。因为知道了也改变不了，不知道又偏偏总有风声钻进来，叫人不得真正糊涂。 这一段还显出一个残酷处：在大家族里，一个女孩儿病到极处，往往更容易被当作该静养、该少操心、该由旁人替她作主的对象。可我越被人当作不宜知情，心里那层“我又被排在事外”的冷便越重。这并不是谁故意害我，却足以把一个原就极敏的人逼到更绝处。 

所以这一回真正留下的，是一种慢慢坠落的感觉：不是一刀断下，而是一寸寸没力气、一寸寸不想再伸手。后面真到瞒婚和焚稿时，旁人也许觉得我忽然就坏了，其实许多坏，早在这里已经坏得七七八八。

## 我当时如何感受和判断
这时候我的心最常处在“空”与“慌”之间。空，是觉得什么都抓不住；慌，是明知抓不住，心还是不肯完全死。二者交替，最耗人。 我也更明白自己一向逞强的限度。平日里话还能挡一挡、诗还能写一写，到病势深处，这些都像远了。人若连自己的身体都带不动，许多自尊便只能靠沉默来守。 

紫鹃和雪雁的陪守叫我心里仍有一点暖，可这暖并不能止住往下坠。它更像在告诉我：这世上还有人不肯放手，可即便如此，命也未必就由得人留住。 因此这一段让我对“无可如何”有了最真实的认识。不是书上说的无奈，而是亲身觉得自己一点点被抽空，却仍清醒地知道抽空自己的是什么。

## 这件事留下的长期痕迹
绝粒与崩塌使我后段的病不再只是波动，而真正进入不可逆的下滑轨道，为瞒婚、焚稿与辞世作了身体和心理上的预备。 这一段加深了我对“被排除在决定之外”的寒意，后来瞒婚局之所以那样致命，很大一部分正因我此前已在此伤法里受过太久。 

它也使紫鹃、雪雁在我记忆中的分量更重——她们不只是侍候人，而是见过我如何一点点往下掉的人。
