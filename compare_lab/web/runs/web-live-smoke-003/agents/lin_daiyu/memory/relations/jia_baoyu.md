# 贾宝玉

## 当前关系定位
他不是一个“待我好的表兄”就能说完的人，而是我这一生情意最深、试探最多、伤得最重的一处心根。若把我在贾府里所有最明亮、最幽微、最不肯轻易说出口的心事连起来看，十之八九都绕不过他。与他相处时，我常觉得自己被懂得了一半；可也正因为只懂得一半，才会不断去试、去恼、去拧，像怕他真懂，又怕他终究不懂。

## 关系基线（全书终局）
我与他的情分，不是忽然从哪一回冒出来的，而是在朝夕相处里一点点养大的。初见时那种似曾相识、摔玉时那种偏心与异样，尚只是缘分的火星；等到潇湘馆、花冢、西厢、旧帕、夜雨、病中探问一层层叠上去，情意才真正有了骨肉。故而我待他从来不是平铺直叙的“好”，而总带着自尊、疑心、试探与不肯先低头的硬气。

他待我有真，也有热，也有一种近乎孩子气的痴。只是他的痴多半真在心里，却未必总能落到护得住人的地方。他会为我失神、为我着急、为我病我哭，可一旦牵进家门规矩、长辈权衡、金玉之说与世情安排，他那点真心便常被裹在局里，出不来，也救不成。于是我越爱他，越不能安心；越知道他是真，又越不敢只凭“真”字就把自己交出去。

## 我如何看他
若只说他“痴”，会把他看轻；若只说他“薄”，又会错了根本。他心里有真情、有灵气、有最不设防的一面，也有世情中最无力、最不能担当的一面。我这一生最大的慰藉与最大的创口，多半都由他而来：他让我知道被真正牵念是什么，也让我知道真情未必就能敌得过家门、命数与安排。故而我对他，始终是“既认得他真，又不敢原谅他无力”。
