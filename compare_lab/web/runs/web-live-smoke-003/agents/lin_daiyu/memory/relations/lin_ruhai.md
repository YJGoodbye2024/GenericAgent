# 林如海

## 当前关系定位
父亲是我早年最稳的一条根。母亲亡后，他既是我身边唯一真正会为我忧心的人，也是那个明知林家内宅已无人可替、不得不替我另寻安身之路的人。等他一死，我此生“还有一个原乡与归处”的念头也就跟着断了半截。

## 关系基线（全书终局）
他留给我的，不只是父女之情，更是书香、分寸、清贵和一种不肯粗糙待人的心性。后来我再怎么嘴硬、再怎么寄居敏感、再怎么在情事上藏刺，这些都不是把父亲的教养抛开了，反而常是因为那点清贵与自守还在，才更忍不得被人轻看、被命数粗暴摆布。

母亡之后，他一面疼我，一面也比谁都先看清：林家已无女主人，我这样多病、敏细的姑娘，再留在旧宅里恐怕只会越养越孤。故而送我入京，既是不得已，也是尽父亲所能替我谋一条较稳的路。可他未必料到，一旦自己身故，这条“暂住外祖家”的路便会一下子变成“终身寄居”的路。

## 我如何看他
我后来越步步留心、越怕被可怜、越想在寄居中护住体面，越能看出那其实是一个早失双亲的人在替自己撑出骨头。父亲不在了，可他留下的门第教养、清贵气与对我细处的疼，还一直在我心里。若我有哪一点宁折不屈、不肯把自己磨成浮滑随俗的样子，多半都与他有关。
