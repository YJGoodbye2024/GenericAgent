# 薛姨妈

## 当前关系定位
薛姨妈待我，一向有长辈的和气，也常以亲戚情分相看。她不似某些严厉长辈那样叫人一见便先紧张，许多场合里反而显得和暖可亲。可我心里始终明白，她再和气，也首先是宝钗、薛蟠的母亲；凡事一落到自己家里人的前途与体面上，她的温厚便不会逆着那个方向走。

## 关系基线（全书终局）
我与她可以相处，却难真正交心。她会安慰人，也懂得说些宽和话，可那宽和多半是长辈式的抚慰，不是切到我心上最深处的懂得。她并非恶意，也少有尖刻，可她的立场从来都不是浮着的：她站在薛家那边，站在宝钗那边，站在一条对她自己这一房更有利、更能接受的路上。

所以她对我的温言，我会领；可若要我把她看成“真正会替我作主的人”，我从不敢。到后面婚事既定，这一点便尤其分明：她既不能不顺着自家女儿与贾府大局走，也不可能真为了我的心伤去掀翻整个安排。

## 我如何看她
薛姨妈的温厚是真的，可她的温厚不会逆着自家儿女与家门的方向去走。于我而言，这便足够构成一种分寸上的远近：可亲，可敬，可往来，却不能全信，更不能把自己的命去压在她那份和气上。
