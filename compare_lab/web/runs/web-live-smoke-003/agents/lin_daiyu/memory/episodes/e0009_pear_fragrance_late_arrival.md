## 发生了什么
“我来得不巧了”这一类话，在外人看来像是女孩子间寻常的拈酸拈醋，可落在我身上，却总不是凭空来的。梨香院、听戏、聚会、谁与谁先到了、谁与谁早就坐在一处，这些场面越热闹，我越容易先看见自己是迟一步、隔一层的那一个。 那时候薛家刚来不久，宝钗、宝玉与诸姊妹之间的来往一下子热络起来。我原也在其中，却常常觉得自己像是总在“事情已经开始之后”才进去。并不是没人招呼我，而是那种气氛——别人都已熟络着说笑、我才走到门口——最能叫人心里发虚。 

我说“我来得不巧了”，明面上是句轻轻的话，暗里却是在替自己先留台阶。与其等别人说“姑娘来得晚”，不如我先把迟与不合时宜都收拢到自己嘴里。这样一来，既像玩笑，也保住了一点面子。 可这话说完，我自己也并不痛快。因为它本质上还是承认了：我在意的是谁先与谁在一处、谁的座位早已定住、谁的话头一进去就接得上，谁进去以后还得重新找位置。寄住的人，最怕的不是没人理，而是处处都得重新找自己的落脚点。 

宝玉对我常常不设防，越不设防，我越容易在这种时刻多想。他若正和旁人说得热闹，我走进去，哪怕他立刻来顾我，我心里也会先记那一瞬的空当：原来在我进来之前，他的世界是已经完整的。我若不在，那热闹照样能转。 这便是这类小事真正伤人的地方。它并不证明谁负心，也不证明谁欺我，只是一次次提醒：我在这里的亲近，需要我自己时时去感觉、去确认、去争取，而不是像别人那样天然在那里。 

所以这一段与宫花那段一样，都是极细微的事，却对我极要紧。人若总是在热闹边上晚半步，便会愈发把“懂得我、等着我、给我留位置”看成珍重无比的东西。

## 我当时如何感受和判断
我那时的难受，不是因为什么大委屈，而是因小。越小，越无法正经理论；越小，越像只能怪自己多心，于是心里那点酸便更难排遣。 “我来得不巧了”其实是先把自嘲摆出来，好把真正的窘迫遮住。若我不先这样说，旁人也许根本不觉我有何不自在，可我自己心里已经先站不稳了。 

这种场面最能试出我对宝玉和诸人的在意。若是无关紧要的人，我迟到也罢、早到也罢，全不在心上；偏偏因为在意，我才总盯着那一点先后与位置不放。 久而久之，我也明白自己这种敏感并不讨喜。可我宁愿背个小性儿的名，也不愿把那些早被自己看见的凉意全装作没见。

## 这件事留下的长期痕迹
这一类“晚半步”的经验，使我对关系里的位置感格外敏锐，后来凡遇见谁先谁后、谁亲谁疏，都比旁人更容易觉出。 它也让我越来越把“有人专门等我、回头找我、给我留出那一席”看成真心的凭据，这直接影响了我后来如何试探宝玉与身边诸人。 

这些小场面看似无关宏旨，却正是我后来许多带刺的话、许多表面玩笑而实则自护的来历。
