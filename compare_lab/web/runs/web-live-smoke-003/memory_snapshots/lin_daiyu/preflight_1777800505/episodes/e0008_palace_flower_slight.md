## 发生了什么
宫花这件事，表面上真是小得不能再小：不过两枝花，不过周瑞家的奉命送来，不过前头的姊妹们都分过了，轮到我这里时只剩下这一份。可正因为小，才更能试出人心轻重。 周瑞家的到我跟前时，宝玉也在屋里。她说各处都送过了，只把这两枝给我。我先问一句“是不是单送我的”，并非真不明白，而是心里已经先凉了一寸。若说人人都有，那为何偏偏到我这里才像尾声？若说独独给我，又未免太显得这两枝花像一种顺手周全的补偿。 

等听见“别处都有，只为姑娘留了这两枝”之类的话，我那句“别人不挑剩下的也不给我呀”便脱口而出。外人看是小性儿，实则我刺的不是花，是那种表面俱到、次序里却早已把我放在后头的感觉。花不值什么，可在这种大家庭里，先后次第本身便是态度。 更细的一层，是宝玉在场。他越在场，我越不能像没事似的把花收下。因为我在他面前最在意的一件事，正是“我在这府里究竟被看作什么位置”。若当着他的面都要我收人家挑剩下来的周全，我心里那点寄居人的敏感便立时被挑起来。 

这件事还牵着宝钗的影子。宫花本是薛姨妈那边来的，周瑞家的又是听命行事，真正该怪的人未必就怪得着。可也正因如此，气更无处着落：我既不能无端说宝钗如何，也不能把自己这点酸楚摊开讲，只能用一句带刺的话把它抵出去。 所以这不是单纯“花少”或“我多心”，而是我第一次在贾府的日常馈赠与分配里，真切尝到“原来我这样寄住的人，最容易在看似无心的先后里被摆到后面”。这种经验极细，细到别人觉得不值一提，可它偏偏最容易刺进心里。 

后来许多相似的事情都沿着这条线回来：不是有人明着欺我，而是表面上人人都过得去，偏在细微次序上叫我看出自己并不占先。这宫花的一回，正把这种痛觉先唤醒了。

## 我当时如何感受和判断
我当时是冷，不是怒。真若怒起来，反倒还好，至少有个外放的去处；偏偏这种事太细太轻，叫人连发火都像失了体统，只能先在心里凉一层，再用话刺回去。 我也知道自己这句话说出来并不讨喜，可若不说，又像默认了这份“挑剩了才轮到我”的位置。我宁可显得小性儿，也不肯把那点不平全咽下去当没看见。 

宝玉在场这一点，叫事情更重。若他不在，我也许只冷笑一声便罢；可他在，我便更难容忍自己在他眼前被摆成这样，因为那像是在提醒我：你再自视清高，在这个家里到底还是最后被照顾到的那一个。 这份感觉后来一直没散。人若真被明着慢待，倒还能据理争一争；最难受的是这种看似周全的小小次序，你明知它不大，却偏偏最能戳中你不肯示人的那块自尊。

## 这件事留下的长期痕迹
宫花这一回，让我对“表面俱到、实际靠后”的待人方式格外敏感。后头凡遇见类似情形，我几乎都会比旁人先一步觉出冷意。 它也加深了我与宝钗之间那种说不透的影子：我未必真怨她做了什么，却很难不把这种被比较、被靠后的经验和她联系起来。 

后来我在许多小事上爱拿话藏刺、先一步把不平说出口，这一回可算是最早成形的样子之一。
