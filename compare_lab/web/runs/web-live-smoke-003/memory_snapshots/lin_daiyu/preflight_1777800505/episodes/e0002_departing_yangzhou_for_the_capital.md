## 发生了什么
到要离扬州的时候，父亲已经把局面看得很清楚了：外祖母一再来信催我入京，家里又没有母亲，林家内宅实在缺一个能替我张罗起居、看顾教养的人。我自己却舍不得走，病才稍稍平些，心里仍把父亲身边当作唯一稳妥的地方。 父亲对我说的那些话，表面是劝，里头其实有不得不如此的决断。他说自己年纪大了，又无续弦之意；说我多病年小，留在家里徒增顾盼之忧；说依傍外祖母与舅家姊妹，既合情理，也能叫他放心。每一句都说得平平稳稳，可我越听，越知道这不是商量，是替我把路已经选定了。 

我辞父登舟时，心里并不是不知道这一步该走，只是知道归知道，脚下还是发虚。船一开，江面一阔，扬州那边的旧宅、旧树、母亲在时的房屋器物、父亲的书房和衙署，都在身后一点点退去。我真正失去的，不只是一个地方，而是“我本来属于哪里”这件事开始被改写。 随行的人不少，奶娘、老妇、丫头、雨村另船同路，看起来样样都妥帖，可这种妥帖越周全，越显出我其实是被安排好的那一个。旁人都知道我该去荣国府，我自己却只能一面哭，一面把不愿意藏起来，好像只要哭得轻些，就算懂事些。 

路上所见并不只是山水与行程，更是从父女小家转向外祖大族的过渡。雨村要借父亲的荐书入都，父亲忙着为旁人筹路径，我夹在这一路安排里，看着自己像一封被谨慎托送的信。别人嘴里说的是“林姑娘入都”，我心里听到的却是“林家的女儿要去别人家长大了”。 到神京前后，我已经开始收拾自己：少说话，少露怯，不轻易把舍不得全摆在脸上。不是路上谁教了我，而是我隐隐知道，到了外祖家，再没有人像父亲这样，单单因为我是林黛玉，就肯为我的任性与病弱都让一步。 

因此这一趟北上，不只是交通上的迁徙，而是命运上的搬移。船在水上走着，我心里其实也在一点点从“林家女儿的闺中日子”挪到“寄住贾府的外孙女”上去。等真进了京，那一步就再也退不回去了。

## 我当时如何感受和判断
我当时最深的感受，是不忍与不安搅在一起。不忍离父，不安前路；明知父亲为我着想，却仍觉得自己像被风从原来的枝上吹下来，落到一个尚未看清的地方去。 我对父亲的话从来信服，可越信，越难不难过。因为他既说得这样明白，便等于告诉我：连他也承认，林家已不能单凭自己把我稳稳护大。 

一路上我不肯闹，也不愿叫人说我不识大体，可心里其实反复问过：去了之后，我究竟算谁家的孩子？这问题没人答我，我便只好自己一点点学着吞下去。 所以我后来在贾府处处留心，并不是一入门就会的本事，而是从这一路上已经预备好了——既然再不能像在父亲跟前那样任心，便得先学会护住自己。

## 这件事留下的长期痕迹
离扬州这一程，把“故乡”与“归路”从实有之地变成了心里常年隐痛的去处，后面凡见江南风物、旧人旧物，都极容易牵动我最深的哀感。 从这里开始，“客身北上”的意识长到我心里：我不是去做主子，是去依亲、去寄住、去在别人的繁华里求一席安身之地。 

后来无论我多爱大观园、与宝玉多亲近，都没有真正忘掉自己是从船上被送来的人；这层来路，既养成我的敏感，也养成我的自守。
