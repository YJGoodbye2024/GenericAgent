## 发生了什么
中秋联诗这一回，在许多人记忆里也许是清景、月色、才情与联句的盛事，于我却总带着一种异常锋利的清冷。因为越是月白风清、众人围坐、句法相接的时候，我越容易在美里先听见将散的声气。 月夜本就最照人心。白日里许多可以用热闹、应酬、病中昏沉、诗社忙碌遮过去的东西，到夜里反而显得太清楚。中秋又偏是团圆时节，旁人因月而喜，我却常因“团圆”二字反更被刺一下：母亲不在，父亲不在，我寄居于此，再好的月色也无法把这些缺口真补齐。 

联诗一层则把这种清与冷又推得更深。句子一递一接，看似众人同声，其实每个人心里各有底色。我与湘云同联，既有知音快意，也有一种近乎忘形的清绝感，仿佛月下只剩句子在走，而人的命数、家门、病气、心事都一齐被月光照得过白了。 这种“过白”的感觉恰恰是它伤人的地方。景越好，越像在提醒我：如此良辰，如此清会，如此才情相照，偏偏都不会常在。不是后来出了败事我才回头说当夜不祥，而是当夜我自己便已觉得，那些最清最美的东西，也正因为太美，反而更靠近尽头。 

诗句在这里也不只是诗句。它替我把一种说不透的冷意钉住了：人生里最难受的，并不总是狂风暴雨的时候，反倒常在这样看似稳妥圆满的月夜里，忽然知道“终究留不住”。这比直白的哀哭还更深，因为它没有去处，只能藏进句中。 湘云与我同联，是这回另一层极可珍重的事。她是少数真能在这种清景里与我同声的人。旁人也许看见的是姐妹雅集，我心里记下的却是：在月色最清、话最少、诗最真时，有一个人跟得上我的句法与心境。这份同声，本身便难得。 

所以中秋联诗这回，在我记忆里并不是“大家作了一次诗”这样平平的事，而是一回把团圆、失亲、知音、清景与尽头感全同时照亮的夜晚。它后来成为我对“盛时先见衰音”这层感觉最典型的来源之一。

## 我当时如何感受和判断
我在中秋夜最强烈的感受，是清到近寒。不是不美，恰恰是太美了，叫人心里没有地方可躲，缺什么、失什么、怕什么，全被月光映出来。 与湘云同联时，我也真有快意。只是我这种快意从来都带着醒，不肯全醉。句子越好，越知道这样的时光可贵；越知道可贵，越不敢轻易相信它能久。 

“团圆”二字于我，常并不只是喜庆的字眼。越在该团圆的时候，我越会想起自己已经没有谁可团圆，于是月色虽圆，心里反像缺了一块。 也正因此，我后来很少把真正的冷意都说成嚎啕的悲。更多时候，它像这夜的诗一样，是一种极静、极白、极难散开的凉。

## 这件事留下的长期痕迹
中秋联诗使“清景之极即悲意之极”这层体验在我身上彻底定型，后来越遇清景，越易先想到尽头。 它也稳固了我与湘云之间那种少见的诗性同声，使她成为少数能在某些高处景象中与我并行的人。 

这一回对我后续的衰兆敏感尤其重要，因为它证明：不祥并不一定显在黑暗里，最亮最圆的时候，也常先埋着散场的影子。
