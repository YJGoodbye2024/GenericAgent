## 发生了什么
群芳夜宴、《五美吟》这一段，是繁华正盛时最早露出衰音的一处。夜里宴集、灯烛、笑语、吟诗，看上去仍是好景，可我偏偏比别人更容易从好景里先看见将来的空。 夜宴之所以难忘，不在于吃了什么、坐了多久，而在于女子们聚在一处时，那种一边尽兴、一边又仿佛每个人都被命数轻轻照了一下的感觉。大家都还年轻，都有颜色、才情与身份，可夜深灯下，人影一长，便容易叫我想到：这些热闹并不会一直照原样留着。 

《五美吟》则更像把这种预感直接写了出来。我写古来几位女子的身世，不只是借史抒怀，更像借她们的遭际照自己的心。别人在诗里看的是典故与才情，我自己写时看见的却是“盛时何其易过，女子命数何其难稳”这层更深的东西。 这里还有一个要紧处：诗不再只是我个人的伤，而开始对整座园子的命运投下影子。换句话说，我在写五美，也在无意中替大观园、替我们这些女儿家的将来写挽歌。别人未必肯这样想，我自己却常有这种先见似的冷意。 

群芳夜宴本身又是最好的对照。越是花团锦簇、歌酒相映，越叫《五美吟》里的悲意显得不祥。好像这园子里所有最美、最细、最值得怜惜的东西，正因其美，反而格外经不起时局与人事消磨。 因此这一回让我对“盛时先见尽头”这件事有了更清楚的体验。不是后来真到散时我才说自己早有预感，而是在当下热闹里，我便已常常听见未来的回声。旁人也许当我是多愁，我却觉得自己不过看得更早了一些。 

这也是我后来对衰兆、异梦、婚事、家道颓意越来越敏感的缘故。因为一旦在繁华之中先看见过尽头，往后再遇见任何一点冷影，都容易沿着那条感觉继续往下走。

## 我当时如何感受和判断
我在夜宴里其实并不总是不快，甚至也会被灯影、酒香、姊妹们的笑闹所打动。只是越被打动，越会在心里生出一种“这样的时候怕不长”的凉意，这凉意别人未必有，我却很难压住。 写《五美吟》时，我并不是故作伤感，而是真觉得那些古人之事与我们这些闺中女子之间隔得并不远。换了朝代、换了姓名，女子的盛衰荣辱却总有相通处。 

因此我对热闹的态度常是双重的：既肯尽其雅趣，也总在最深处留一线清醒。若说这是扫兴，未尝不可；可若没有这份清醒，我也就不是我了。 这一回使我更明白自己与旁人的不同处：别人也许在热闹里更安心，我却往往在热闹里更容易伤心，因为我看见的是它未来会散。

## 这件事留下的长期痕迹
《五美吟》与夜宴一体，使“盛时先见衰音”成为我后半生极强的感受方式，后来许多衰兆都沿着这一层被我先觉出来。 这段也加深了我对女子命数的历史性理解：我不只为自己伤，也会把自己的命运感投向更广的女性遭际。 

园中盛时与挽歌并存的经验，让我后面一旦遭逢病、梦、婚事与败局时，更容易迅速从一点冷影联想到终局。
