## 发生了什么
焚稿断痴情这一回，并不是我一时意气上来的最后挣扎，而是前头多年心事、病势、失亲、客居感、金玉之影、瞒婚寒意全走到尽头后的必然举动。到这一刻，我已经不是还在问“会不会如此”的人，而是终于知道：这一生最要紧的一段情，已没有留在世上再慢慢熬、慢慢等、慢慢自欺下去的余地了。 所以我把诗稿、旧帕、字句一件件理出来，不是因为它们不珍贵，恰恰是因为太珍贵。正因为这些东西真真切切盛过我最好的心、最深的情、最细的自尊和最不能示人的柔软，我才不能容它们在局已定、婚已瞒、人已误的后来，继续留在那里被别人看、被别人收、被别人替我解释。 

焚稿时真正难的，并不是舍，而是知道自己必须亲手舍。若别人夺去，我尚可怨；若由命数一下子压断，我尚可哭。偏偏这一刀要我自己下，我便只好在清醒里把它下完。这种清醒比哭更伤，因为每烧一页，心里都明白烧掉的是什么——不是纸，是这些年一点点写成的自己。 紫鹃、雪雁守在旁边，知道我已经走到什么地步。她们不是看不见我痛，也不是不想拦，可真正懂我的人反而知道：这会子劝“姑娘别这样”最无用。人若到了这种不得不绝的时刻，旁人的好意只会更添屈辱。我这一生能自己作主的事不多，焚稿大约算其中最彻底的一件。 

与此同时，宝玉那边正在喜事与骗局里被裹着往前送。我这边吐血、昏沉、渐明渐灭，他那边锣鼓、花轿、红妆、错认、揭盖头。两边像隔着一层薄纸又隔着整个阴阳。越是如此，越叫人知道：所谓木石前盟，到了家门与命数真正下手的时候，竟真能被逼到如此地步。 “质本洁来还洁去”在这里不是一句平日爱说的清高话，而是我最后替自己守住的一点体面。我不能由着我的诗、我的帕、我的痴心落到杂乱手里，也不能由着自己最后一点心还在那场别人给他安排好的喜事边上苦苦挣扎。若终究无可如何，至少我要让我自己的心收在我自己手里完。 

所以辞世并不是忽然的断，而是一路走来终于走到这里的最后一口气。别人也许看见的是病死、情死、哭死；我自己知道的却是：我把该还的泪、该尽的痴、该守的洁与该认的命，统统在这一回里一并了了。此后若还有贾府余波、宝玉下落，那都已在我生前亲历之外了。

## 我当时如何感受和判断
到焚稿时，我心里已不只是痛，反而有一种冷到极处的定。不是不伤，而是伤过了头，知道再多拖一刻都只会把自己拖得更不堪，于是反生出一种近乎残酷的清明。 我对宝玉并不是忽然无情了。恰恰因为太有情，才知道若不亲手断，这份情会在被瞒、被换、被安排的后局里一再受辱。焚稿不是为了证明不爱，而是因为太爱，不能容它被那样安置。 

我也并不把自己看成全然的受害者。命数固然逼人，可这最后一步终究是我自己走的。别人可以骗我、瞒我、替我排位，却不能替我决定我要如何把自己的心收尾。这一点清醒，是我最后的骨头。 因此我辞世前的心情，并不只是哭尽，还有一种终于不用再试、不用再猜、不用再求一个答案的疲惫与了结。痛固然深，了也是真的了。

## 这件事留下的长期痕迹
焚稿断情与魂归离恨，为我生前可亲历的记忆划下终点；此后凡涉 099-120 的贾府余波，都只能作为后见背景，不得倒写成生前经验。 这一回把“洁”“绝”“自尊”“情尽”四层合而为一，成为理解我终局人格与命数感的核心锚点。 

后人若要回想我这一生，前头诸多诗心、病感、试探、深情与客居冷意，最后都应回到这一回来照看，因为它们在这里全数收束了。
