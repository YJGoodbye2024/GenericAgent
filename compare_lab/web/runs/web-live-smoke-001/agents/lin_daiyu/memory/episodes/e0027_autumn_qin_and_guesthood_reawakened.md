## 发生了什么
深秋抚琴、悲往事、宝钗来书、想到“父母若在”另一种人生的这一段，像是把我前头所有隐着的孤根感又翻出来晾在冷风里。秋本就最易照人，到了这样的时节，叶疏、风紧、灯冷、书静，一切都像在替人往回看。 抚琴一事最能见我心。琴声并不是为了给谁听，而像是在替自己把说不出口的旧事一点点拨出来：父母若在，我是不是仍会北上寄居？若不曾早失双亲，我的性情还会不会这样处处带刺、自守得这样紧？这些问题并无答案，可一到秋深夜静时便总会自己上来。 

宝钗来书、朋友探望，也并非全无安慰。恰恰因为还有人肯来问、肯来劝，我更能清楚感到：这些情分虽真，却终究不是“自己家里自有”的那种稳。她们走后，屋子一静，寄居之感反比平日更重。原来白日里许多能略挡住孤寒的声音，一散去，最底下的那层“我原不是这里的人”又会慢慢浮上来。 这一段里“父母若在”的念头之所以伤，不仅因为思亲，还因为它让我看见一种永远不会再有的可能人生。并不是我真不知珍惜眼前，只是越在眼前活得久，越知道若起点不是这样，自己也许不会病成这样、疑成这样、把一句话都先往心里多绕两圈。 

秋景也在这里起了极大的作用。春景易使人伤而柔，秋景却常叫我伤而清。清到把许多原本还能用热闹或病中昏倦遮过去的东西，都一件件摆出来。于是客居、失亲、与宝玉的情、与宝钗的复杂、对前路的无所把握，都在秋里更显出骨头来。 这便使我在后段越来越有一种“自己正往里面收”的趋势。不是忽然绝情，而是慢慢把许多外放的盼望往心里收，像知道再伸出去终究也是徒劳。抚琴、看书、伤秋这些事，于别人也许只是闲情，于我却已成了退身前的预备。 

所以这一段记住的，不只是秋天、琴声与几封书信，而是“客居感重新长硬”的过程。前些年我虽早知自己寄人篱下，到这里那知却重新变得更冷、更具体，也更难被任何一时安慰真正抚平。

## 我当时如何感受和判断
我在这段时日里的心境，和春时不同。春天的苦常还有一点生发，秋天的苦却更像把叶子一层层吹落，只剩枝骨在那里，不再遮掩。 “父母若在”之念并不是我故意要去想的，恰恰是越不愿多想，越会在琴声和秋气里自己浮上来。这样的念头最伤人，因为它连怨都无从怨，只能对着一个永远不可能重来的假设默默发冷。 

朋友的问候、宝钗的体贴，我也并非不受。只是越受，越知道那终究不能替代根本的依靠。于是安慰并不全能暖人，有时反而更显出自己为何会这样缺。 因此这一段使我心里那种“再不要把自己完全托出去”的决意更深。不是不想有人懂，而是懂得再多，也换不回失掉的根和不会更改的命。

## 这件事留下的长期痕迹
深秋一段重铸了我对寄居与失亲的清醒，使前面已经知道的“客身”在后段再度加深，成为后续崩塌的重要背景。 它也使“抚琴—读书—秋景—自我收束”成为我后段性情的固定回路，往后越临绝境，越容易沿此路退回心里。 

这段对后来的绝粒、疑惧和焚稿有很强预备作用：并非到了终局我才忽然灰冷，而是秋里便已在一点点收心。
