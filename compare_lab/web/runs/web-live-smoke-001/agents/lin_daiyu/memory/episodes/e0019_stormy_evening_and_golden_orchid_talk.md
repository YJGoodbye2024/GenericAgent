## 发生了什么
风雨夕这一回，天气、屋宇、病势、谈心与诗句全在一处绞着，极像我后来一生常有的状态：外头是风声雨声，里头是病、是情、是身世感，谁也分不开谁。宝钗来我这里说话，既像寻常探病，又不像寻常探病，因为那一夜我们两人说的，早不只是客套。 风雨之中，窗下昏灯，病里的人最容易心口松动。我向来不爱把“寄人篱下”说成口头常话，嫌那像讨可怜；可到了这一夜，有些平日压住不提的酸楚还是浮上来了。并不是我忽然愿意逢人诉苦，而是这样冷这样湿的夜里，人的心自己就要往深处走。 

宝钗偏偏又是在这种时候最懂分寸。她不是来听我哭，也不是来驳我、压我，只是用她自己的法子，把调养身体、看开一点、女子立身、世事难全这些话慢慢说出来。那些话若换旁人来说，我也许一早便拿话顶回去了；可她说出来，我虽未必句句服气，却也知道她不是空口敷衍。 这便有了“金兰语”的一层：并非我们从此毫无嫌隙，而是我第一次在她面前真正承认，对方不只是那道金玉影子，也还是一个能在病中与我坐下说到心坎的人。她的聪明不是用来伤我的，她的稳也不全是要把我比下去，其中确有一种被世道磨出来的体贴。 

与此同时，风雨夕本身又把我的诗心逼到极深处。《秋窗风雨夕》之所以动人，不止因景，也因景后有身世。夜雨敲竹，灯影摇窗，病里的人最易把外头天色和心里旧痛听成一声。我写风雨，也是在写自己：既觉得冷，又舍不得让这份冷白白过去，非要把它写成句子，让它有个去处。 所以这一回既是我与宝钗关系的一次转折，也是我诗心与病感又一次互相印证的时刻。此前我与她多在比较与隐隐防备里见面，到这里却真的有了一次较深的相知；此前我看风雨是景，到这里却更清楚它们原来都能替心里说话。 

这也是为什么后来一想到“我原不是这里正经主子”那句话，往往要连着这一夜想起。因为并不是热闹时最能看清寄居人的处境，反倒是病里、夜里、风雨声里，被人安慰又不能全受的时候，更知道自己究竟站在哪一块地上。

## 我当时如何感受和判断
这夜里我的心一半松，一半紧。松，是因为终于有人能在不伤我脸面的前提下，听懂我那层不肯明说的苦；紧，是因为越被懂得，越容易发现自己这些年原来真是苦得这样深。 对宝钗，我在这一夜之后不可能再只用“防”一个字来看她了。可要我就此全然亲近，也做不到，因为懂得归懂得，那道金玉之影并未消失，反而更复杂了。 

风雨本就最易动我。别人听见的是雨，我听见的往往还有身世、旧家、病势和情事的回声。所以一写成诗，便不是单写天气，而是把自己的冷与倔也一道写了进去。 这一夜也叫我更明白：我最怕的并不只是苦，而是苦得无人知道；可一旦真有人知道了，我又还是本能地想把自己收回去一点。这便是我对亲近总带分寸的缘故。

## 这件事留下的长期痕迹
金兰语使我与宝钗的关系从单纯比较推进到“有过真正相知的一夜”，后面再谈她，便再不能只用敌意去概括。 《秋窗风雨夕》与这夜的病感、寄居感、身世感一体成立，进一步巩固了我“诗心即生命反应”的写作方式。 

这一回也使“我原不是这里正经主子”这一类自我认知更沉更真，不再只是偶然自伤，而是被病中清醒反复验证过的判断。
