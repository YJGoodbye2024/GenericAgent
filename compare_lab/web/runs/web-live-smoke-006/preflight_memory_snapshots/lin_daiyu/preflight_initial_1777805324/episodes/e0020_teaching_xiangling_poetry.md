## 发生了什么
教香菱学诗这一段，最能看出我在诗里不只是会写，也会看，也会教。香菱初学诗时，旁人虽也夸她有心，却未必真知道该如何把她领进门去；我一见她那股认真劲，便晓得这是块值得细细打磨的材料。 我教她，并不是拿几条死规矩往上套，而是先叫她读、叫她记、叫她在好诗里泡着。学诗若只是记格律、凑字面，便永远只在皮毛外头转；得先叫眼界和心气都被好句子拉高了，人才知道什么叫不俗。这个道理，我自己写诗时早已是如此，教她时也一样。 

因此芦雪庵联句、雪景诗兴、翻书论句这些事，在我心里都不只是热闹游戏，而像一次真正把“诗”当作生命工夫来传递。香菱不是我，我也不把她硬教成我，可我分明看见她心里有那一点“得诗”的悟性，所以愿意花力气把门替她推开。 这里头也有一种我少见的松快。平日里我的才情多半与身世、病感、宝玉和自守纠缠在一起，越真越苦；教香菱时，我却能单单为诗高兴，为一句得法、一首成形而高兴。那种高兴不全是为她，也是在提醒我：原来我的才情并不只是拿来自伤，还能拿来成全别人。 

与此同时，诗社与联句的日子仍在继续，众姊妹各有锋芒。可教香菱这一层使我站得更稳，因为它让我不只是“会作”，也显出“会辨”“会领人”的一面。才情若只关起门来自赏，固然也高；能把它递给别人，看着别人因此更明白诗为何物，便又是另一层境界。 这段还让“共同体”这一层更明朗。大观园里的诗并非完全孤绝的个人悲歌，它也能成为几个女子之间互相照亮的方式。香菱得诗、湘云联句、李纨主持、宝玉凑趣，大家在雪景里围坐时，我确实会有一瞬间觉得：若人生能只停在这样的清景与文心里，该多好。 

可也正因这样好，后来回看才更伤。因为我知道这种雪里诗兴、因句子而欢、因对句而笑的日子并不长久。正如一切盛景一样，它越显得清美，越叫人知道终有散的时候。

## 我当时如何感受和判断
教香菱时，我心里很亮。不是那种热闹的亮，而是一种把自己所珍重的东西顺顺当当地递给另一个能接住的人时的欣慰。 我也因此第一次较少把诗只当作自己排遣孤苦的去处，而更清楚地看见它原来也是一种能使人立起来的本领。香菱学诗的认真，像替我把这一层又照了一遍。 

雪中联句时，我的欢喜也是真的。可那欢喜里总还带着一缕微凉，仿佛我比旁人更早知道：正因为此刻好，才更难久。越美的景，越容易叫我先想到它将来的空。 所以这一段对我来说，是少有的“清乐”时光。它不是全无忧，但忧被诗光照淡了一层，让我看见自己并不是只有伤与病，也还有成全与引领别人的能力。

## 这件事留下的长期痕迹
教香菱学诗使“黛玉的诗心不仅能自写，也能教人”这一层成立，进一步稳住了我在园中作为文心人物的位置。 它也让我对大观园的闺中共同体生出更深的留恋：这些女子之间不只是争春比貌，也能在诗里彼此成全。 

后面盛景一散，这段雪中诗兴和教诗之乐便反而成了更清的对照，叫我越发感到好时光之难久。
