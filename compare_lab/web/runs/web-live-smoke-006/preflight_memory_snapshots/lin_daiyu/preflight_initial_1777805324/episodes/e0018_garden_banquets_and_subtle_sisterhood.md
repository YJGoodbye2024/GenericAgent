## 发生了什么
园宴、清茶、栊翠庵、诸般小聚这一段，表面看是大观园繁华正盛时的闲雅生活，细看却很能照出闺中女子之间那些不必嚷破、却日日都在流动的理解、比较与分寸。于我而言，这不只是热闹可记，更是我与宝钗、妙玉、众姊妹之间几种不同关系都渐渐坐实的时候。 在这种聚会里，贾母一来，众人说笑有序；凤姐忙着周全，丫头婆子各司其职；姐妹们看似同席，其实每个人的姿态都不一样。有人适合应酬，有人适合收拾场面，有人适合出句行令，有人则在景与物里更容易显出真意。我在这种场合里常不抢热闹，却也不愿真把自己藏没了，总要在该说一句时说一句，把那一点清峭劲儿留下。 

宝钗在这里与我相处的方式尤其耐人寻味。她并不总与我针锋相对，反而常常在清茶细物、病中调养、闲语往来里，把许多不好直说的话说得恰到好处。她懂分寸，也懂退让，甚至懂得怎样在不伤人脸面的前提下叫场子圆过去。这种本领我看得见，也不得不服。 妙玉则又是另一种镜子。栊翠庵品茶、雪水、器皿、洁净、口味，这些看似小玩意，在她那里都有过于讲究的一面。我与她气味相投处在于，我们都厌俗、爱净、对清物有近乎执拗的偏心；不同之处则在于，她像是把自己往更孤绝的地方收，我却仍旧被人情和情爱牵着，不能全然抽身。 

因此这一段虽然并无一件最惊心的大事，却把我在闺中共同体中的位置照得更清楚：我不是最圆融的，也不是最会周旋的；可在清景、诗心、细物与真情这些层面，我总有自己的尺度，旁人即便不全认同，也不能说没看见。 更重要的是，这些细雅往来叫我开始真的“懂”宝钗，而不只是防她。她替我解癖、论调养、说世情，未必句句都合我心，却不是全无真意。我并不因此就放下心结，可至少明白了：她并非只站在我对立面，也有她自己被世道磨出来的一套温存与克制。 

所以这一段值得记住的，不在一席宴、一杯茶，而在我作为黛玉，在贾府盛景的最细处里，看清了自己与诸人之间那些不必大起大落、却足以决定以后如何相待的气脉。

## 我当时如何感受和判断
我在这种盛时宴饮里常有一种双重感：一面确实能从清景细物里得趣，一面又总像先听见散场的回音。越是雅，越知其难久；越是周全，越觉其背后有许多不能尽说的用心。 对宝钗，我在这时候开始有更多“懂而不肯先软”的心。她有她的好、她的稳、她的照拂，我并不是看不见；只是我若全认了，便像承认她那一套活法也许比我更合世道，这一点我始终不甘。 

妙玉则叫我看见另一种极端的清。她近似我，却又比我更绝。看着她，我会想到自己若把人情和情字都割去，是否也能那样冷净；可我终究知道我做不到，因为我这颗心太会被牵动。 因此这些盛时小聚里，我虽也笑、也作陪、也赏景，心里其实时时都在辨人、辨气、辨自己究竟更像谁、离谁更近、又绝不肯变成谁。

## 这件事留下的长期痕迹
这一段加深了我与宝钗之间“既防且懂”的复杂关系，也让妙玉的清绝、众姊妹的不同气质在我心里各自定了位。 它使我更明白自己在闺中共同体里的位置：我能以诗心、清峭与真意立住，却绝不是靠圆融取胜的人。 

这些微妙的姐妹相知与相较，后来都成了我看人处世的重要底色，使我的孤高不只是自闭，也带着比照之后的选择。
