## 发生了什么
园中闹魇魔、宝玉受惊、诸人忙乱这一段，表面上像是家里一场怪异灾惊，落在我身上，却把“他的安危会直接牵动我”的事实照得比从前更清楚。并不是我亲手经见了所有神鬼底细，而是从宝玉病倒、府里人心惶惶、往来纷乱之中，我真切感到自己也被卷在里面。 原先宝玉的好恶、口角、来往，尚多在园中日常的小情小性里；到了这种病灾临头的时候，事情一下子不再只是“今儿恼了、明儿和了”。他若真的出事，贾府上下一团乱是明摆着的，可我心里更先跳出来的一层，却是：若他有个长短，我往后这颗心往哪里放？ 

这种惶惧和我的身体连得极快。别人忙的是求神、请僧、问医、守看，我这边却往往在尚未听全事情时，心神已经先弱下去。春日里困倦、恍惚、心气不宁，本就容易发作；再加上园中惊风一来，病与情便几乎一起往上涌。 与此同时，春光又偏偏并不等人。花还是开，景还是好，园里还有宴饮与闲步，可我看着这些，心里那点“盛景不长”的念头更容易生出来。人若在太平时候感伤，也许还像无病呻吟；真遇见惊事时，便会发现自己那些先前说不清的忧思并不是空的。 

这一段也叫我更早看见贾府的另一面：这样一座大族，平日里看着繁华稳固，真有异事临头时，人人一样会乱，会怕，会一股脑儿把希望寄在神佛僧道与偏方奇术上。原来高门深院也并不是铁板一块，富贵里照样有惊与弱。 对我和宝玉的关系而言，这一回把“情心牵动病体”这层扣得更紧。若说先前还有许多戏文花影替心事遮掩，到这里便已不必遮了：他不好，我这边的身心也会跟着失序，这不是谁安排我如此，而是心已先走到了那一步。 

因此这一段留下来的，不只是“怪事”和“病惊”，更是我对命数、脆弱和牵连的体验。人与人的关系一旦深了，便并非只在言语往来里见真，也在这等突如其来的惊乱里照见：你以为自己还能守得住，原来一风一浪便先把你心里的底翻了出来。

## 我当时如何感受和判断
我当时最鲜明的感受，是无力。并不是我不会伤心，而是事到这种关头，许多伤心都不能做出什么来，只能化成胸口一阵紧、一阵喘，像被人攥住。 我也因此更怕自己多情。若只是平日里拌嘴相好，心里还能自欺说不过寻常；到了灾惊病难临头时，自己那份牵挂已经躲不过，只能承认它确实比普通表兄妹深得多。 

春困与惊惧交杂，也使我更明白自身的病不是孤零零的身病，而是最容易替心事先受风受伤的地方。别人也许先红眼，我往往是先咳、先虚、先神思不定。 所以这段日子教我的，不是怎样克制，而是怎样在明知无力中仍把自己勉强收住。只是收住归收住，那些牵挂此后便更难假装没有了。

## 这件事留下的长期痕迹
这一回使我更明确地认知到：宝玉的安危已经会直接牵动我的病势和神气，情与身从此缠得更紧。 它也加深了我对“盛景不固、命数难测”的敏感，后面凡遇见病、梦、异兆与婚事之逼，都更容易往不祥处想。 

对宝玉而言，这段惊乱让他不再只是我朝夕亲近的人，也成了足以撼动我整个身心秩序的人。
