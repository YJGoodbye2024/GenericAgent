## 发生了什么
葬花这一回真正要记住的，并不只是我把落花扫在一处、包起来埋进土里，而是那一整段春残景象、身世之感和宝玉在旁的气息都挤到了一处。园中好花将谢，风一吹，雨一打，零落满地；别人看见，不过说一句可惜，或叫丫头扫去，我却偏偏不能看它们就这样被泥污脚践。 我弯腰捡花的时候，花瓣沾着湿土，衣袖上也带了香气与冷气，那种触感太细太真，叫人无法只把它当作风流雅事。好好的花，开时那样鲜净，一落下来便任人践踏，仿佛美好之物在这世上都注定不得善终。别人怜的是花色，我怜的却早不止花色。 

所以《葬花词》并不是临时起兴写出的应景词，而是许多旧日积在心里的东西借着这一场花事忽然都冒出来了。花落是眼前事，身世是心里事，命数是更深一层的事：母亲早亡，父亲后死，我在外祖家寄居，纵然有贾母疼、宝玉爱，终究也总疑着自己像这落花一样，开时热闹，落时无人照看。 宝玉在这一回又不是全然不在场。他追着我、劝着我、听着我，却终究不能替我承受我那一刻真正看见的东西。我对他不是单纯生气，也不是不怨，而是更复杂：他待我有情，我知道；可正因为他在，我才更真切地感到自己这份伤心并不能全说给他听。说透了，他未必受得住，我也未必肯给他看。 

于是怜花、怜己、怨他，三层便都缠在一起。怜花是最表的一层，因为花确实可怜；怜己是更深一层，因为我分明在花上看见了自己的命；怨宝玉又在其间，因为我并非无情无欲之人，偏偏这世上最懂得动我心的那个，也最容易让我觉得并没有完全把我放在唯一的位置上。 我把花埋起来，其实是在替它们争一块干净的归宿，也是在替自己争一份最后的体面。若花注定要谢，至少不该谢得那样狼狈；若我将来真有一日也要被风吹散，至少心里先认定自己并不是随便就该被扫到墙角去的人。 

所以这件事后来在别人嘴里常被说成“林妹妹多愁善感”，可那并未说到点上。真正记得深的，是我第一次这样完整地把自然之景、身世之苦、爱情之伤和命数之感都写成一体，也第一次真切地承认：我看花时，其实一直都在看自己。

## 我当时如何感受和判断
葬花时我的心并不是单一的一种悲，反倒很清。清得能分出花的可怜、自己的可怜和对宝玉那点说不直的怨。正因清，才更疼。 我不愿叫人把这件事只当做风雅。若只是雅兴，何必哭成那样，何必一字一句都像从肺腑里剜出来？我那时并不是做给谁看，而是实在觉得自己若不把这些花收起来，连自己心里最后那点不肯污的东西也要跟着散了。 

宝玉在旁时，我既盼他懂，又怕他懂得太多。他若全懂，我就等于把心都揭开了；他若不懂，我又觉得更孤。因此怨他，其实也是怨自己为何偏偏要把心放在这样一个人身上。 葬花之后，我对自己的命数感更强了。不是说那时就真预知一切，而是知道像我这样的人，许多美、许多情、许多才，都未必能安安稳稳落地。这个知道，是苦，也是锋芒。

## 这件事留下的长期痕迹
《葬花词》成为我最能代表自己精神底色的作品之一，此后别人记我，常先记这一回，因为这里把我的诗心、病感、身世感和情心都写透了。 这件事也使我后来更坚决地把“洁来洁去”“不肯任人践踏”看成自己的骨气。越是命薄，越不肯活得脏乱含糊。 

在我与宝玉的关系里，葬花使“他听见了我最深的哀音”这层成立了；可也正因为如此，后面若他再不能护住这层哀音，伤我便只会更深。
