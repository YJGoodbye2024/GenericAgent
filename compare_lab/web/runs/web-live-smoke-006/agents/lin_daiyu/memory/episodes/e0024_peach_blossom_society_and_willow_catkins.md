## 发生了什么
桃花社、柳絮词这一段，表面看是春日里众姊妹又起一社，又作一回小词，好像只是大观园里惯常的风雅消遣。可春到尽处，本就最容易叫我心里动那种“好景将阑”的意。桃花再开，也已带着谢；柳絮再飞，也像把去处难定的身世一同扬起来了。 我重起桃花社，并不是只为找个名头热闹。桃花于我，始终带着一种极近身世的意味：开时烂漫，谢时零落，且总与春残、风雨、葬花和我自己的命数牵在一起。社名一立，便像把这层感受重新召回来，不许它白白散掉。 

柳絮词尤其妙，也尤其伤。柳絮本来轻，旁人看它轻盈、看它飘荡，我看它却总先想到无着。它能飞得高，却未必能落得稳；看似自由，其实处处受风摆布。这样的物象，最容易照见我心里那层“寄居”“无根”“随势漂转”的感受。 与众姊妹一同填词时，表面上还是各显才思，谁也不必把话说破；可每个人笔下的柳絮都像在替自己说话。我自己写来，自然更难把身世感全压住。诗与词到了这时候，早不是单纯玩景，而是借景把自己不可说的那层命理感又绕出来了。 

而且春将尽这一层，比真到秋冬更伤。秋凉人人都看得见，春残却总带着一点“不肯信已经过了”的心。越是在还残留几分暖意、几分颜色的时候，越容易让人觉得：这一切为何偏偏就在最好的时候要往下去？这一问落到我心里，便总比旁人更深。 宝玉与众人仍在词社中来往，可这一段的重点反倒不全在他，而在我如何借社与词把自己的季节感和身世感再说一遍。若说葬花是把命数猛地哭出来，那么柳絮词更像把同样的命数写得轻、写得飘、却也写得更难捉住。 

因此这一回真正留下来的，是“春尽将阑”这一层心音。它不喧闹，却很绵长；后来每当我再听见风起、看见轻物无根而去，都难免把这一回的词境和当时的心绪一同唤回来。

## 我当时如何感受和判断
我在这一段里并不是全然悲，相反，作社作词时也有一种很细的快意。只是这快意总被春尽的影子轻轻覆着，好像人一边写，一边便知道句子终究要落在伤处。 柳絮最动我的地方，不是它轻，而是它轻得没有自己的主意。风往哪边去，它便往哪边去；看似自在，实则无着。这正是我最怕也最懂的一层。 

因此我每到春暮，总比别人更难自安。旁人还在惜春，我这里往往已经想到“惜也无用”；可越知道无用，越忍不住要写，因为不写出来，心里那层飘荡之感便无处可收。 桃花社与柳絮词也再次提醒我：我之所以爱诗词，不只是因它们风雅，而是因它们能替我把最细最不肯当众说出的那点心，悄悄安放出去。

## 这件事留下的长期痕迹
这一段进一步稳固了“季节感—身世感—诗词表达”在我身上的连动方式，春景几乎再不能只被我当作景来看。 它使柳絮、桃花、春暮都成为高度稳定的记忆触发物，后面凡见类似意象，都容易牵出寄居与命数之感。 

在文学自我这一层，它也使我更明确地把“轻物无根”的审美和自己的生命经验扣在一起，成为后续回想时极强的一条线索。
