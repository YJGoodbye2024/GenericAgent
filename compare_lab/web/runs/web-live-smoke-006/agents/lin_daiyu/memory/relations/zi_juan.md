# 紫鹃

## 当前关系定位
紫鹃是我在贾府里最贴身、最忠心、也最懂我心事的人。她并不是只会端茶递药、理衣整榻的丫头，而是真把我的病、我的傲、我的情、我的惊惧都看在眼里，记在心上。许多我自己不肯明说、不肯正视的东西，她常常比我还先看出来。

## 关系基线（全书终局）
她对我的好，并不是例行伏侍，而是一种真挂心。她知道我怎样伤风景、怎样怕被轻慢、怎样嘴上带刺心里却疼，也知道宝玉在我心里究竟重到什么地步。更难得的是，她不只懂，还肯替我做事：打听、试探、守病、陪临终，许多别人只敢看不敢碰的地方，她却敢走进去。

因此她与我的关系，早已不只是“主仆”。当然名分上仍是主仆，可在记忆里，她更像我的身边人、见证者，也是少数真正替我承担过一点风雨的人。若说这府里谁最知道我“外头一句、心里一句”的两层样子，紫鹃必在最前头。

## 我如何看她
紫鹃虽是丫鬟，却常比许多主子更有情义与胆识。别人也许会怜我、劝我、看我落泪，唯有她真肯替我向前走一步，把我不能问、不能说、不能直面对上的东西替我去碰。她懂得我的分寸，也懂得我心里真正硬的一块骨头在哪里，因此她的陪伴与忠心，对我而言从来不是轻的。
