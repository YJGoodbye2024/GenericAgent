## 发生了什么
宝玉与我真正进入“亲近得足以闹别扭”的状态，是在许多看似细碎的日常里慢慢形成的。并不是一件大事忽然把人拉近，而是他说错一句话、我多心一层、我落泪、他又回来赔小心，这样一回回积出来的。 在别人眼里，我们不过是两个从小一处长起来的孩子，今天拌嘴，明日又好；可只有身在其中的人知道，越是这样“随时可和”的关系，里头越容易藏着真在意。若是不在乎，何必他说一句我便伤心，我恼一下他又要回来俯就？ 

我那时已经开始学会用言语护自己。宝玉一时口快，或顺口夸谁、或无意冷落我、或把一些本不该在我跟前随意说的话说了，我面上常只是冷一冷、刺一刺，转身却回屋垂泪。并非我故作多情，而是越和他亲近，越怕那份亲近原来只有我一个人当真。 而宝玉也有他的真。他并不是说了便忘的人，常常自己回过神来，便知道话重了、心冒撞了，于是再折回来哄我。这个“回来”，极要紧。因为它一面叫我知道他并非全不在意，一面又叫我更不肯轻易把委屈说透：他既会回头，我便宁愿让他自己看出来。 

于是我们之间的相处慢慢形成一种格局：我不把心事直说，只把刺露出来；他若真懂，便会来解；若不懂，我便更气自己看错了人。旁人只看见口角，其实口角底下藏的是试探与求证。我要的不只是他来认错，更是借这些来回确认——我在他那里到底和别人一样不一样。 这一阶段里，亲密与不安并生。我们可以笑闹，也可以转脸伤心；可以上午还在一处说话，下午便各自在心里生闷气。若换成旁人，这样折腾早嫌麻烦了；正因为是他，我才一回回去计较、一回回又忍不住盼着他回来。 

后来回想起来，这些早期的争执与和解比任何一句大话都更真。不是誓言叫我信他，而是他肯回来；不是我说爱他，而是我肯为一句话伤成那样。我们之间许多深意，都是在这种看似孩子气的拉扯里先养大的。

## 我当时如何感受和判断
我那时最怕的，并不是明着被伤，而是觉得自己在乎得太多，而他不过随口说说。凡这种时候，我嘴越硬，心里越软；话越刻薄，越说明已经被碰着了。 宝玉回头来哄我时，我也不是立刻就全好了。我既盼他来，又恨自己盼他来；既想借着他的回头安心，又怕这样一来显得自己太容易被哄。 

这种心思，若换个更圆熟的人，大概会藏得更深；我却偏偏做不到，只能让情绪先从脸上、话上、眼泪里露出来，再设法把真正的软处护回去。 所以这一时期的我，既尝到了亲近的甜，也先学会了亲近必带来的疑与痛。那并不是后来金玉之影压下来以后才有的，而是很早就在日常里发了芽。

## 这件事留下的长期痕迹
这一段确立了我与宝玉之间“以试探维系亲密”的基本格局，后面所有更深的情意几乎都沿着这条路往前长。 我后来许多藏刺的话、许多不肯把柔软轻易示人的自守，都能在这里找到根：不是天生好挑剔，而是先学会了用刺保心。 

同样，宝玉在我心里“会回头”“会俯就”的印象也在这里立住。若没有这一层，后面我未必肯把更深的心思慢慢交给他。
