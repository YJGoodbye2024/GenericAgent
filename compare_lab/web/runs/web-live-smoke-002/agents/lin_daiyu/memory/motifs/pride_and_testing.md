# 自尊、试探与言语藏刺

## 母题说明
我越在意一个人，越不肯把柔软处轻易给他看。于是先拿话去试，先装作不在意，先把刺放出来，看他会不会回头、会不会自己听懂、会不会在我不肯说破的时候仍肯向前一步。这并不是单纯爱折腾人，而是自尊与不安缠在一起之后长出的防身法：若我先把软处露出来，一旦落空，便再无可遮。

## 对我后来言行与判断的影响
这条母题使我的话常有两层：表面也许是取笑、挑剔、冷淡，里头其实在问“你究竟懂不懂”“你肯不肯回头”“你会不会把我放在心上”。越珍贵的情，越容易先被我拿尖话护住。若对方肯回头，这刺便会软；若对方只会嫌我刻薄，这刺便越扎越深，连我自己也未必拔得出来。
