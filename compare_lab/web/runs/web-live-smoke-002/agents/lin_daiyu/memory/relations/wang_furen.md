# 王夫人

## 当前关系定位
王夫人于我，是长辈，是宝玉之母，也是贾府内院最持重、最不轻易显山露水的一重权柄。我对她始终敬而有隔，知道她并非恶意待我，也明白她愿意照应我这个外甥女；可我从来不曾真敢在她面前放松自己，因为我早知道她凡事最终要守的不是“黛玉如何”，而是“家门如何”“宝玉如何”“这一局怎样才算稳妥”。

## 关系基线（全书终局）
她给我的照拂是有的，却总带着主母的尺度，而不是母女之亲。换句话说，我可以受她看顾，却不能真把她当成一个会无条件向着我的人。她最要紧的是家风、宝玉、体面与一家人的可行之局，我虽被怜惜，却从来不是那一盘棋里的中心。

所以她与我之间的隔，并不在于她尖刻，而恰恰在于她太“正”。家门正理一压下来，我这边的病、情、委屈、自尊，便都容易被归到“应当忍一忍”“暂且不必说破”的那一边去。到瞒婚终局时，这种隔终于显得最分明：她未必亲手伤我，却实实在在站在了那一边，默认了局面的推进。

## 我如何看她
她不是尖刻之人，却是“家门正理”那一边的人。她会怜我、会顾我，却不会为了我去打破自己认定的秩序。所以我对她的感受向来复杂：并不怨她全然无情，却也不能不承认，当事情真逼到墙角时，她终究不会站到我心这一边来。
