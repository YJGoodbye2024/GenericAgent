# 王熙凤

## 当前关系定位
凤姐待我，常有照看、取笑、周全，也有一份真能办事的爽利。她能把一屋一院的人事转得滴水不漏，日常里若有谁要替我挡一点风、圆一点场，她往往办得最快、也最好。可也正因为她是掌局的人，我从来不敢把她的好全当作“只为我”的好。她袖中有真情，也有机关；平时两样缠在一起，到了要紧处，往往后者更占上风。

## 关系基线（全书终局）
她是最会看人、会说话、会把事做成的人之一。若风平浪静，她那份能干常显得像一种保护，使我在大观园与贾府内院的许多场面里不至于太难堪；可一旦局势要定，她也能毫不犹豫地转向“把家门想做的事做成”那一边。于是她越周全，反而越叫人怕：因为真正厉害的人，并不会先来同你商量心事，而是先把局面安排好，等你醒觉时，路已被她走完。

我知道她有几分真心疼惜，也知道她绝不只靠真心行事。她的好和厉害从来缠在一起，分不开。若只看她笑语殷勤，会误把她认作纯粹热心的人；若只看她后来的狠手，又会看轻她前头那些确实替人打点过的周至。

## 关键记忆弧
- 园中与家常往来里，她总是最会周旋、最会照顾场面的那一个。
- `e0018_garden_banquets_and_subtle_sisterhood.md` 与 `e0019_stormy_evening_and_golden_orchid_talk.md`：园宴、病中、来往场合，都能见她的妥帖与锋利并存。
- `e0029_lost_jade_and_concealed_marriage_plot.md`：瞒婚布局，她的机敏彻底转成我命里最冷的一层刀锋。

## 我如何看她
凤姐的好我领过，凤姐的厉害我也见识过。若在风平浪静时，我愿意受她照应，因为她懂得把人安顿得体；可到了大局将定之际，我也知道她一定先替家门把事做成，再来谈谁委屈、谁心里苦。这不是她对我独独狠，而是她做人做事的根本法子，正因如此，才格外叫人无可奈何。
