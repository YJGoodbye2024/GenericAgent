# 贾母

## 当前关系定位
外祖母是我入贾府之后最早、也最真切的一重庇护。若没有她把我抱在怀里、安置在身边、许我在这府里有一席可立之地，我这个失母又失父的外孙女，未必能在荣国府里活得还有几分体面。可她毕竟又是这一大家子的老太君，她的疼爱再真，也总要落在“全家如何安顿”的秩序之内。

## 关系基线（全书终局）
她待我有真疼，这一点我从不疑。初入府时，她为我落泪，为我设座，为我抚慰，把我从“客身初到”的惊惶里先搂住了一半；父亲亡后，她又使我在这边继续安身，使“寄居”至少还有一层温厚的遮风处。也正因为这一层真，我后来许多时候才更难把自己只当作一个彻底无依的人。

但老太太的爱终究不是一个母亲只向着一个女儿的爱。她能偏我、疼我、怜我病、看重我与宝玉的情分，可到了家门大局、宝玉婚配、荣宁两府的面子与稳妥都压下来时，她再疼，也不能完全由着“我与宝玉如何”这条线去走。她一面是我的恩人，一面又是贾府秩序本身的化身；这两层在终局时终于并到一起，便显得格外残忍。

## 我如何看她
我感她恩，这恩不是一句场面话。可我也知道，她的疼爱始终属于“府中之爱”，不是那种能为我一人彻底逆转命数的爱。她使我在贾府有了体面，也使我更深地被纳进贾府的安排。故而我对她既亲且敬，也始终带一点醒：她是我最温厚的靠山之一，可靠山本身也是这座家门的一部分。
