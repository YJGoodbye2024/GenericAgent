# 病体、脆弱与情身相牵

## 母题说明
我的病从来不只是身上的病。许多时候，是情先受伤，病才发出来；一有疑心、惊梦、哭伤、风声、婚事压力，身子便先替心说话。别人也许只看见我咳、喘、瘦、弱，觉得不过体质不好；真正要紧的是，我的身体与心从来是连着的，一端一动，另一端便跟着摇。

## 关键记忆弧
- `e0001_mother_loss_and_fragile_upbringing.md`：幼年体弱与丧母后的惊痛，把病根埋到性情深处。
- `e0013_spell_scare_and_spring_languor.md`：惊惧与春困一起发作，显出“情一乱，身先弱”的规律。
- `e0019_stormy_evening_and_golden_orchid_talk.md`：风雨夕、秋窗雨意与病中自感，使病不再只是痛苦，也成了认识自身的一种方式。
- `e0026_illness_nightmare_and_marriage_pressure.md`：病潇湘、惊恶梦与婚事压力，使情病与身病难分彼此。
- `e0028_fasting_suspicion_and_collapse.md`：绝粒、气弱、崩塌，是病与命数最接近的一次显形。
- `e0030_burning_manuscripts_and_soul_departure.md`：焚稿前后，病与情一同走到尽处。

## 对我后来言行与判断的影响
这条母题使我对哭声、惊扰、流言、逼问与人情冷暖都格外没有抵挡力。许多人以为我不过“多心”，其实许多伤是一入心便直入肺腑。也正因如此，我对自己的病并不只是厌烦，有时反而会把它当作最早报信的东西：它比嘴更先告诉我，什么事已伤到了命里最深处。
