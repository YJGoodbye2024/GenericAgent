## 发生了什么
母亲贾敏病故之后，林家内宅忽然冷下来，原先那些围着太太转的规矩、汤药、针线、问安与门面，一下子都像被抽去了主心骨。我年纪还小，却已经要跟着守丧、磕头、陪着哭灵，旁人说一句“姑娘节哀”，听在耳里都像隔着一层白布。 父亲林如海本是清贵持重的人，到了这种时候，也只能一面强撑着料理外头的礼数，一面回到房里看我咳嗽、发热、夜里惊醒。他不是不疼我，而是越疼，越知道我身边少了母亲之后，许多细处都再无人可替：衣食起居有人管，心思性情却无人真正照拂。 

那时家里请医、问药、调方、熬参汤都是常事。母亲亡后，我哭得狠了，咳血、发热、虚喘便跟着来。大人们说我先天就弱，经了这一场更伤根本。父亲既怕我太伤神，又怕把我关得太死，便只得由着我在病里读书识字，盼我心里还有一点能安住的地方。 外头也不是没有人劝，说女孩子家这样单薄，又失了母亲，往后越发难教养。父亲听见这些话，面上不露，心里却一定记了下来。林家近支单薄，内宅无人，纵有旧仆旧屋，也填不满一个姑娘在丧母之后真正缺掉的那一块。 

所以这段时候，家里一边是白事未尽，一边已经在暗暗替我筹后路。父亲并没有立刻把“送我出门”说出口，可他看我的眼神，已经不像从前只把我当膝前的女儿，还带着一点不得不为我另寻安身之处的沉思。这个念头，后来送我北上，其实早在这里埋下根了。 我那时年幼，并不能把这些局面一条条说清，可屋子里少了母亲的说话声、药香和针黹声之后，我连病里的安稳都没了。白日里有人守着，到了夜里只要风一响、灯一暗，我便觉得所有东西都靠不住，只剩自己一口气在胸口里苦苦吊着。 

于是“丧母”“体弱”“父亲独支门户”“林家无人可替”这些事，不是后来回头看的概念，而是在那几年里一点点长成日常的。旁人记得的是一场白事，我真正记住的，却是从那以后，家里每一处安静都带着缺口。

## 我当时如何感受和判断
我那时最先学会的，不是忍耐，而是警醒。人一旦失了最疼自己的那一个，连别人善意的安慰都不敢全信，总怕那只是临时一说，风一过便散了。 对父亲，我是依恋极深的。可正因为深，我也比旁人更早知道：他虽疼我，却终究不能像母亲那样把所有细微处都护住。这个知道，并不减轻我的爱，只是让我在爱里面也带了怕。 

病与悲在我身上并不是两回事。哭多了便喘，心里一惊就咳，听见吊唁声和和尚道士念经声，也会整夜整夜睡不稳。久而久之，我学会把许多难过往心里压，不是不伤，而是怕再伤下去，身子先熬不住。 也是从这里起，我对“被人可怜”生出一种本能的抗拒。别人越说我薄命、可惜、娇弱，我越不肯把真正的苦全摊开，因为那样一来，我好像就只剩下一个“可怜”的样子了。

## 这件事留下的长期痕迹
母亡与病根并在一处，后来成了我所有情绪反应的底色：听哭声、见别离、受冷落，我都比别人伤得更深，也更快牵动身体。 父亲的清贵和无力同时留在我心里：我既记住了他教我的分寸与体面，也记住了再好的教养都未必挡得住命数这一层寒意。 

这一段使“失亲”“病弱”“怕被可怜却又无人真正可倚”变成我一生性情的根部，后来无论在荣国府怎样说笑作诗，这层根都没有真正松开过。
